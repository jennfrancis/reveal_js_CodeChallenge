Getting Started with Git

Git lets every team save and share their work without overwriting each other.
Follow these steps once at the start of the challenge.

Step 0 — Check Git is installed

Open a terminal and run:

git --version

You should see a version number. If not, download Git from https://git-scm.com/downloads and run the installer, then come back here.

Step 1 — Clone the repository

This downloads all the challenge files to your computer.

git clone https://github.com/<org-or-user>/reveal_js_CodeChallenge.git
💡 Replace <org-or-user> with the GitHub username your mentor gives you.

Then move into the project folder:

cd reveal_js_CodeChallenge

Step 2 — Create your team folder

Each team works in their own folder — no spaces in the name, use hyphens.

mkdir teams/team-<your-team-name>

Then copy the starter files into it:

cp quiz.html quiz.css quiz.js hints.md teams/team-<your-team-name>/
📁 Work only inside your team folder from this point on. Never edit the files in the project root.

Step 3 — Save your work (end of session)

Run these three commands at the end of every session:

# 1 — Stage your changes
git add teams/team-<your-team-name>/

# 2 — Commit with a message
git commit -m "Day 1: completed HTML structure and CSS styling"

# 3 — Push to GitHub
git push origin main
Write a short, honest commit message — future-you will thank you!

Step 4 — Pull at the start of each session

If you're on a different computer, or picking up where a team-mate left off:

git pull origin main
Always pull before you push to avoid conflicts with other teams.

Daily workflow — quick reference

# ── START of session ──────────────────
git pull origin main

# ... edit files in your team folder ...

# ── END of session ────────────────────
git add teams/team-<your-team-name>/
git commit -m "Short description of what you did"
git push origin main

Troubleshooting Git

Permission denied when pushing
git config --global user.email "you@example.com"
error: failed to push (rejected)
Someone else pushed first — run git pull origin main then push again.
fatal: not a git repository
Wrong folder — run cd reveal_js_CodeChallenge first.
Accidentally edited the wrong file
git diff to see changes, then git checkout -- <filename> to undo.
🆘 If you're stuck on Git, ask a mentor before running any other commands.

Step 3 — Quiz Logic JS ~35 min

Open quiz.js. This is the most important step.
You'll write JavaScript to show questions and check answers.

🎯 Goal: Clicking a button highlights it green/red and advances to the next question automatically.

3a — The questions array

An array is a list. Each item is an object with three properties.

const questions = [
  {
    question: "How many planets are in our solar system?",
    answers:  ["7", "8", "9", "10"],
    correct:  1          // index of the right answer (0 = first)
  },
  {
    question: "What is the closest star to Earth?",
    answers:  ["Sirius", "Betelgeuse", "The Sun", "Proxima Centauri"],
    correct:  2
  },
  {
    question: "On which planet is the largest volcano?",
    answers:  ["Earth", "Venus", "Jupiter", "Mars"],
    correct:  3
  },
  {
    question: "Approximately how old is the universe?",
    answers:  ["4.5 billion years", "13.8 billion years",
               "1 billion years",   "100 billion years"],
    correct:  1
  },
  {
    question: "What force keeps planets in orbit around the Sun?",
    answers:  ["Magnetism", "Friction", "Gravity", "Electricity"],
    correct:  2
  }
];
💡 Add your own questions later in Step 5!

3b — State variables & grabbing elements

let currentQuestion = 0;
let score = 0;

const questionText  = document.getElementById("question-text");
const answersDiv    = document.getElementById("answers");
const scoreDisplay  = document.getElementById("score");
const questionArea  = document.getElementById("question-area");
const resultArea    = document.getElementById("result-area");
const resultText    = document.getElementById("result-text");
const restartBtn    = document.getElementById("restart-btn");

document.getElementById() finds an HTML element by its id attribute so we can change it with JavaScript.

3c — showQuestion()

function showQuestion() {
  const q = questions[currentQuestion];

  // Update the question text
  questionText.textContent = q.question;

  // Clear old buttons
  answersDiv.innerHTML = "";

  // Create a button for each answer
  q.answers.forEach(function(answerText, index) {
    const btn = document.createElement("button");
    btn.textContent = answerText;
    btn.classList.add("answer-btn");

    btn.addEventListener("click", function() {
      checkAnswer(index);
    });

    answersDiv.appendChild(btn);
  });
}

3d — checkAnswer()

function checkAnswer(selectedIndex) {
  const q = questions[currentQuestion];

  // Disable all buttons so the player can't click again
  const buttons = answersDiv.querySelectorAll(".answer-btn");
  buttons.forEach(function(btn) { btn.disabled = true; });

  if (selectedIndex === q.correct) {
    score++;
    scoreDisplay.textContent = score;
    buttons[selectedIndex].classList.add("correct");
  } else {
    buttons[selectedIndex].classList.add("wrong");
    buttons[q.correct].classList.add("correct"); // show right answer
  }

  // Wait 1.2 seconds then move on
  setTimeout(nextQuestion, 1200);
}

3e — nextQuestion() & start

function nextQuestion() {
  currentQuestion++;

  if (currentQuestion < questions.length) {
    showQuestion();
  } else {
    showResult();
  }
}

// Kick everything off!
showQuestion();
Save and refresh. Can you answer a question and see the correct one highlighted? 🎉

Step 4 — Results Screen JS ~30 min

Show the player how they did at the end, with a personalised message and a "Play Again" button.

🎯 Goal: After the last question the quiz area hides and a result screen appears with the score and a message.

4a — showResult()

function showResult() {
  // Hide questions, show result area
  questionArea.classList.add("hidden");
  resultArea.classList.remove("hidden");

  const total = questions.length;
  let message = "";

  if (score === total) {
    message = "🌟 Perfect score! You're a space genius!";
  } else if (score >= total / 2) {
    message = "🚀 Great job! " + score + "/" + total + " — keep exploring!";
  } else {
    message = "🌍 " + score + "/" + total + " — revise and try again!";
  }

  resultText.textContent = message;
}

4b — restartQuiz()

function restartQuiz() {
  currentQuestion = 0;
  score = 0;
  scoreDisplay.textContent = 0;

  questionArea.classList.remove("hidden");
  resultArea.classList.add("hidden");

  showQuestion();
}

restartBtn.addEventListener("click", restartQuiz);
Finish all questions. Does the result screen appear? Does "Play Again" reset everything?
